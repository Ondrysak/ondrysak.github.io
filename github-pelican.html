Pelican with Github

Published: Po 07 květen 2018
Updated: Po 07 květen 2018
By Ondřej Naňka

In Technology.

tags: python pelican github

Hello fellaz, I am sure you would love to make a wonder page like this one using Pelican and Github. Let's do it!

You will need a Linux machine with python and pip and a regular Github user account.

STEP BY STEP

First you need to create a repo named YOUR_GITHUB_USERNAME.github.io

Clone it to your local linux machine

git clone https://github.com/YOUR_GITHUB_USERNAME/YOUR_GITHUB_USERNAME.github.io.git
#mine was like this
#git clone git@github.com:Ondrysak/ondrysak.github.io.git

Go into the cloned repo

cd YOUR_GITHUB_USERNAME.github.io

Create a new branch as we want to preserve master for the html output of Pelican

git checkout -b pelican

Install python packages 

pip install pelican markdown ghp-import

Run pelican quick start

pelican-quickstart

Go to content folder and create a file first.md in your favorite editor

Modified: 2018-05-07 09:01

Hello fellaz, if u did not hear about love its cool!

To generate html and try the web on localhost go to back to the folder with the generated Makefile

make html 
make serve

Add, commit and push the changes in the source to the pelican branch

git add . && git commit -m 'first commit to da branch yo' && git push

Push the generated html to master using Makefile

make github

Say woohoo and checkout your new page at YOUR_GITHUB_USERNAME.github.io!
